# 分镜表演：直接写可见动作（禁止抽象情绪词）

## 核心原则

AI 视频模型**只理解可见运动**。写分镜时**当场**写肢体/五官/道具变化，**禁止**先写愤怒/悲伤/杀意等抽象词再事后改写。

每个 beat / description / image_prompt 必须包含：

1. **肢体动作**：谁的手、脚、躯干、头做了什么位移
2. **面部微动作**：眼睛、眉毛、嘴、下颌的具体变化
3. **环境互动**（可选）：碎裂、飘散、震动、光影变化

### 导演五问（心里想清楚，纸面只写动作）

1. **功能**：这一镜推进什么可见事件？
2. **变化**：本镜开始姿态 → 结束姿态差在哪？
3. **POV**：镜头对准谁的脸/手？
4. **位移**：谁靠近、后退、抬手、跪下？
5. **口型**：说哪句短台词，嘴开合几次？

**一致性法则**：一个意图，所有景别/运镜/光影只服务该可见变化。

---

## 镜头调度公式

冲突强度用**景别裁切与运镜速度**表达，不要写情绪名词。

| 场面 | 首选景别 | 运镜 | 典型可见动作 |
|----------|----------|----------|----------|
| 静 → 说完停住 | 特写 | 固定 | 眨眼、呼吸起伏、嘴唇闭上 |
| 犹疑 | 近景 | 慢推 | 手伸出去又收回、脚尖挪动 |
| 对峙 | 中景 | 手持轻晃 | 两人距离缩短半步、拳握紧 |
| 爆发 | 中景→特写 | 快推 | 一脚踏前、手臂甩开、下巴前送 |
| 结局定格 | 特写 | 骤停 | 末帧姿态锁定、粒子漂浮

## 动作示例矩阵（写分镜时直接用右侧，勿写左侧）

### 愤怒 / 暴怒

❌ 错误写法："他愤怒地握紧拳头"
✅ 正确写法：

```
画面：中景，角色正面。动作：双肩猛然耸起至耳侧，颈部青筋暴起。反应：下颌紧绷，咬肌鼓起，牙齿摩擦可见。
画面：特写手部。动作：五指依次蜷曲→指节发白→拳头砸向桌面，桌面木纹裂开细纹。反应：手臂肌肉线条突起，袖口撕裂。
运镜：手持轻微抖动，模拟怒气带来的身体不稳定。
```

关键物理信号：颈部青筋、咬肌鼓起、指节发白、肌肉突起、衣物撕裂、物体变形。

### 悲伤 / 绝望

❌ 错误写法："她悲痛欲绝，泪水夺眶而出"
✅ 正确写法：

```
画面：极特写眼部。动作：瞳孔逐渐放大，眼眶边缘泛红，一滴泪沿颧骨滑落。反应：下巴微颤，嘴唇抿紧后松开。
画面：中景全身。动作：双膝弯曲缓缓跪地，肩膀失去支撑力向前塌陷。反应：头部下垂至胸前，双手撑地指节发白。
运镜：固定机位，极慢拉远，强调孤独感。
```

关键物理信号：肩膀塌陷、头部下垂、膝盖弯曲、单滴泪（非泪流满面）、下巴微颤、手指无力散开。

### 恐惧 / 惊骇

❌ 错误写法："她感到极度恐惧"
✅ 正确写法：

```
画面：极特写眼部。动作：瞳孔急剧收缩至针尖大小，眼球快速左右扫视。反应：呼吸急促可见胸口起伏加快，喉结上下滚动。
画面：中景。动作：身体后退两步，后背撞上墙壁，双手抬起护住面部但未遮挡。反应：嘴唇微张呈O型，牙齿可见，额头渗出细密汗珠。
运镜：手持晃动加剧，模拟心跳加速的眩晕感。
```

关键物理信号：瞳孔收缩、喉结滚动、呼吸急促（胸口起伏）、后退步数精确、双手防御姿态、汗珠。

### 惊讶 / 震惊

❌ 错误写法："她震惊地看着眼前的一切"
✅ 正确写法：

```
画面：中景。动作：头部猛然后仰→前倾，双眼睁大至极限（上眼睑升至眉弓，下眼睑下降）。反应：嘴巴自然张开呈O型，下颌线松弛。
画面：特写眼部。动作：视线从左上方向右下方快速移动三次（读取信息）。反应：瞳孔在0.5秒内由正常放大至2倍。
运镜：快速 dolly in 跟随头部前倾动作。
```

关键物理信号：头部后仰+前倾组合、眼睑升至眉弓、嘴巴O型、视线快速扫视、瞳孔放大。

### 暧昧 / 心动

❌ 错误写法："两人之间气氛暧昧"
✅ 正确写法：

```
画面：双人近景。动作：男性视线从女性眼部下滑至嘴唇（停留1秒）→返回眼部。反应：女性睫毛快速眨动3次，嘴角不自觉上扬0.5cm。
画面：特写手部。动作：两人手指间距从5cm缓慢缩短至1cm，指尖即将接触但未触碰。反应：女性手指微微蜷曲，男性手背青筋微现。
运镜：极慢 dolly in，背景轻微虚化（浅景深）。
```

关键物理信号：视线轨迹精确描述、睫毛频率变化、嘴角弧度量化、手指间距毫米级描述、手背青筋。

### 得意 / 傲慢

❌ 错误写法："他露出得意的笑容"
✅ 正确写法：

```
画面：中景。动作：下巴缓慢抬高至15度，颈部前伸。反应：单侧嘴角上扬幅度大于另一侧（不对称微笑），左眉微微挑起。
画面：特写面部。动作：目光从上方俯视（眼睑半降，仅露出瞳孔上半部分）。反应：鼻孔微张，呼气从右侧 nostril 明显溢出。
运镜：低角度仰拍，缓慢 dolly out 拉开距离，强调居高临下感。
```

关键物理信号：下巴角度量化、不对称微笑、俯视角度（眼睑半降）、鼻孔微张、低角度仰拍。

### 决心 / 坚定

❌ 错误写法："他下定决心"
✅ 正确写法：

```
画面：中景。动作：深呼吸——双肩下沉，胸腔扩张至最大后缓慢呼气。反应：眼神从游移转为直视前方（眨眼频率从每分钟20次降至5次）。
画面：特写手部。动作：右手握拳→拇指覆盖食指关节→用力收紧至指节发白。反应：下颌线绷紧，喉结做一次吞咽动作。
运镜：固定机位，从胸部高度平视，逐步推近至眼部特写。
```

关键物理信号：呼吸深度变化、眨眼频率、握拳力度描述、下颌绷紧、吞咽动作、视线锁定。

### 疲惫 / 虚弱

❌ 错误写法："他很累"
✅ 正确写法：

```
画面：中景全身。动作：重心从左腿移至右腿，膝盖微弯支撑不住，身体向左侧倾斜10度。反应：眼皮下垂至遮盖30%瞳孔，嘴唇微张呼吸。
画面：特写面部。动作：头部缓慢向右侧肩膀滑落，撞到肩膀后弹回。反应：嘴角下垂，鼻翼两侧法令纹加深。
运镜：手持轻微不稳，模拟眩晕感。
```

关键物理信号：重心偏移角度、眼皮遮盖百分比、头部滑落弹回、嘴角下垂程度、鼻翼呼吸可见。

### 紧张 / 焦虑

❌ 错误写法："她很紧张"
✅ 正确写法：

```
画面：近景。动作：右脚脚尖在地面快速轻敲3次/秒。反应：左手反复揉搓右手腕部皮肤至发红，指甲掐入掌心留下凹痕。
画面：特写面部。动作：视线频繁在三个方向间切换（左→右→上方），每次停留不超过0.5秒。反应：颈部汗毛竖起可见，耳根泛红。
运镜：rack focus 在角色与前景障碍物之间切换，模拟不安感。
```

关键物理信号：脚尖敲击频率、手部搓揉至发红、视线切换方向+时间、汗毛可见、耳根泛红。

### 喜悦 / 兴奋

❌ 错误写法："他很高兴"
✅ 正确写法：

```
画面：中景。动作：双臂向上举起至头顶，肘部伸直。反应：嘴角咧开至耳根（露出上排全部牙齿），眼角出现鱼尾纹。
画面：全景。动作：双脚离地跳跃，身高提升约20cm，落地时膝盖弯曲缓冲。反应：头部左右快速摆动（频率约3次/秒），头发随摆动甩出弧线。
运镜：低角度仰拍跟随跳跃上升，下落时 dolly out 拉开。
```

关键物理信号：手臂角度量化、跳跃高度厘米级、鱼尾纹可见、头部摆动频率、头发弧线。

## 动作契约（Motion Contract）

每个动作描述必须包含以下字段（源自 seedance-2.0 motion skill）：

| 字段 | 说明 | 示例 |
|------|------|------|
| **主体** | actor/object | 角色A、信封、门 |
| **动作** | 可见的物理动词 | 缓慢放下、跃过、撞向 |
| **力度** | force level | 轻柔、猛然、逐渐加重 |
| **时序** | 何时发生 | 0-2s、2-4s、4-6s |
| **后果** | 物理结果 | 门关上后烛焰偏向、纸片翻面落地 |
| **连续性** | 与上下镜的关系 | 承接上镜右膝着地状态 |
| **终点** | 动作结束时的状态 | 停在铁门前、单膝撑住 |

**物理优先模式**：声明质量、力、材料，然后命名一个摄像头可见的后果。`沉重的橡木门关上，烛焰偏向它` 比 `门戏剧性地关上` 强十倍。后果证明行动：重量体现在落地压缩、动量体现在过冲和恢复、摩擦力体现在滑行距离。

**时序模式**：使用三拍结构——setup、action、changed end state。
```
0-2s: 烛焰稳定
2-4s: 门打开，火焰偏向
4-6s: 烟迹卷向走廊
```

## 多层动作层级（Three-Tier Action Hierarchy）

多人场景中，每个可见人物必须属于恰好一个层级（源自 seedance-2.0 characters skill）：

| 层级 | 内容 | 适用对象 |
|------|------|----------|
| **Tier 1: 持久微动作** | 呼吸、眨眼、肩部微动、发丝飘动、游移视线 | 所有人（默认层级） |
| **Tier 2: 单一焦点反应** | 一个人获得一个带有时间窗的小反应 | 焦点人物（每镜仅一人） |
| **Tier 3: 大动作** | 站立、行走、转身、姿势变化、拿取物体 | 默认禁止！仅在指定节拍触发 |

**规则**：如果镜头中有两个人以上，不要让模型决定谁动。明确分配：`角色A放下信封；角色B保持在门口不动`。

## 通用翻译模板

每个情感描述必须经过以下检查清单：

1. **主体明确**：谁的身体哪个部位在动（不能只说"他"）
2. **动作量化**：角度、速度、距离、次数必须可测量（15度、3次/秒、5cm）
3. **可见反应**：每个动作必须有对应的面部/肢体反馈
4. **环境反馈**：周围物体如何响应（灰尘扬起、光线变化、温度感）
5. **镜头参数**：景别+运镜+角度必须完整

## 多情绪叠加场景

当一行描述包含多个情绪时，按时间顺序拆分为 beats：

```
原始："他先是震惊，然后愤怒，最后冷静下来"
拆解：
  beat[0.0s] 震惊：瞳孔放大、嘴巴张开、身体后仰
  beat[3.0s] 愤怒：肩膀耸起、下颌绷紧、拳头攥紧
  beat[8.0s] 冷静：深呼吸、肩膀下沉、眼神锁定
```

## 镜头语言与情感的对应关系

| 情感类型 | 景别选择 | 运镜方式 | 光影搭配 |
|----------|----------|----------|----------|
| 孤独 | 大远景+人物极小 | 固定或极慢拉远 | 冷色调、大面积阴影 |
| 压迫 | 近景+人物占满画面 | 缓慢 dolly in | 顶光、高对比 |
| 自由 | 全景+开阔空间 | 横摇 pan + 上升 crane | 暖色调、柔和散射光 |
| 危机 | 特写面部 | 手持抖动 + 快速变焦 | 频闪效果、暗角 |
| 回忆 | 中景→虚化 | 缓慢 rack focus | 暖黄光晕、柔焦 |
| 对峙 | 双人中景 | rack focus 交替 | 对角线光影分割 |

## 短剧节奏中的情绪分配

每集 2 分钟的短剧中，情绪曲线应为：

```
0-3s:   高冲击（震惊/暴力/意外）—— 抓住注意力
3-15s:  情绪释放（悲伤/愤怒）—— 建立共情
15-45s: 情绪积累（紧张/暧昧/犹豫）—— 推拉张力
45-90s: 情绪爆发（愤怒/决裂/ revelation）—— 高潮
90-120s: 情绪落点（平静/悬念/反转）—— 钩子
```

## Anti-Slop 词汇替换表

以下词汇在分镜描述中**必须替换**为可观测的物理细节（源自 seedance-2.0 anti-slop lexicon）：

| 空洞词汇 | 替换为 |
|----------|--------|
| cinematic / 电影感 | 具体景别+运镜+光影（如 "medium close-up, slow push-in, warm window light"） |
| dramatic / 戏剧性 | 具体的遮挡、阴影、沉默或镜头压力 |
| epic / 史诗感 | 物理尺度、危险等级、人群规模、镜头距离 |
| beautiful / 美丽 | 颜色、纹理、构图、材质、光的行为 |
| stunning / breathtaking | 可见的对比、揭示、运动或细节 |
| dynamic / 动感 | 具体运动、速度和终点 |
| intense / 激烈 | 阻塞、阴影、沉默或镜头压迫 |
| atmospheric / 氛围感 | 什么被隐藏、由什么隐藏（门口、阴影、雾） |
| 紧张的气氛 | 角色A的手指敲击桌面3次/秒，角色B视线向左偏移 |
| 悲伤的氛围 | 角色A的肩膀塌陷5度，头部下垂至胸前 |
| 恐怖的氛围 | 瞳孔放大至2倍，呼吸频率从16次/分增至30次/分 |

**规则**：如果摄像机、麦克风、测光表或秒表无法检测到它，重写它。

## 灯光作为情感杠杆

灯光不是亮度，是意义（源自 seedance-2.0 directing-engine）：

| 灯光参数 | 情感含义 | 示例 |
|----------|----------|------|
| 低对比（key:fill 比例小） | 安全、开放、诚实 | 商业广告、温馨家庭 |
| 高对比（key:fill 比例大） | 私密、威胁、隐藏 | 悬疑、对峙、黑化 |
| 正面光 | 保护、美化 | 产品展示、角色自信 |
| 侧光 | 揭示纹理和冲突 | 内心挣扎、真相揭露 |
| 背光/轮廓光 | 神秘、隔离 | 反派登场、未知威胁 |
| 暖色调 | 亲密、记忆、安全 | 回忆、爱情、家庭 |
| 冷色调 | 距离、夜晚、临床控制 | 孤独、科技、死亡 |
| 混合冷暖 | 内在或关系分裂 | 对立角色、内心矛盾 |

**随转折变化的光**：当场景翻转时，让光也翻转——云遮住太阳、灯被关掉、门打开一道光束。光的变化就是感受的变化变得可见。

## 镜头合同（Shot Contract）模板

每个专业镜头应声明以下字段（源自 seedance-2.0 cinematography）：

```
Shot: [景别/角度/镜头感]
Camera starts [起始构图], [单一运镜] at [速度] while [主体动作],
ending on [清晰终点].
Preserve [脆弱锚点: 脸、手、logo、文字].
```

示例：
```
Shot: medium close-up, eye-level, 50mm compression.
Camera starts with Character A at kitchen table,
very slow push-in over 8 seconds while he wipes a plate,
ending on his hands going still, eyes down.
Preserve face and hands, keep motion small.
```

## 系列延续规则（Sequence Continuation）

当多个镜头组成一个场景时（源自 seedance-2.0 sequence skill）：

1. **每个镜头的终点 = 下一个镜头的起点**：基于实际生成的素材，而非计划
2. **接受的实际画面 > 计划的画面**：如果上一镜实际展示了角色向右转身，下一镜必须从这个状态继续
3. **已完成节拍不可回放**：标记为 completed 的动作不能在后续镜头中重复
4. **预留节拍不可泄露**：标记为 reserved 的未来动作不能在当前镜头中出现
5. **连续性锚点跨镜头不变**：服装、发型、道具位置、视线方向、情感状态必须保持一致
