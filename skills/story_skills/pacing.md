# AI 短剧叙事节奏

## 流量铁律
- **3 秒抓人**：开篇必须是炸裂画面（惨烈/冲突/悬念），禁止慢镜头铺垫
- **15 秒一小爆**：每约 15 秒须有一个情绪或视觉小高潮
- **结尾强钩子**：定格冲突画面 + 悬念字幕，必留追更

## 2 分钟黄金结构（其他时长等比缩放）
| 时段 | 功能 |
|------|------|
| 0–3s | 炸裂开局，最高情绪/惨烈画面 |
| 3–30s | 事件结果 + 悲情共情 |
| 30–70s | 回忆杀 + 情绪堆叠 |
| 70–110s | 黑化爆发、特效拉满 |
| 110–120s | 定格强冲突 + 悬念字幕 |

## AI 剧本镜头原则
- 剧本层单镜头 **1–3 秒**，特写/近景/仰拍优先（竖屏 9:16）
- 抽象情绪必须改写为可见画面：颜色、裂纹、粒子、光影、肢体动作
- 台词单句 ≤10 字，长句拆句，禁止抒情铺垫

## 情绪切换
AI 无法细腻过渡，用短镜头硬切「极静 / 极炸」，禁止依赖微表情渐变描写。
