# 分镜七要素（硬性）

每镜必须写全 7 项，全部是**可执行可见指令**，禁止抽象情绪词。

| 要素 | 字段 | 写什么 | 短剧常用选项 |
|------|------|--------|--------------|
| 景别 | shot_size | 裁切多近 | 远景 / 全景 / 中景 / 近景 / 特写 / 极特写 |
| 角度 | angle | 镜头相对主体 | 平视 / 仰拍 / 俯拍 / 斜角 / 荷兰角 |
| 构图 | composition | 主体落在画面哪 | 中心 / 三分法 / 对称 / 引导线 / 前景遮挡 / 留白压人 |
| 光影 | lighting | 主光方向与软硬 | 顶光/侧光/逆光/伦勃朗；硬光打脸或软光 |
| 色调 | color_tone | 冷暖与主色 | 冷白、暖黄、青冷、品红对比；高对比短剧色 |
| 动势 | motion | 机位+主体怎么动 | 定镜 / 推近 / 拉远 / 横摇 / 跟移 / 手持微抖 / 环绕 |
| 转场 | transition | 接到下一镜 | soft dissolve / fade black / wipe / match cut / hard cut |

## 写作纪律
1. 七要素服务本镜**一个可见事件**，不要空写“电影感”。
2. `camera` 由「景别 + 角度 + 动势」合成，勿再塞散文。
3. 竖屏 9:16：特写优先，主体落在安全区内；构图写清脸/手的位置。
4. 动势只写**一条**主运动路径（一个推镜或一次主体迈步），禁止堆叠多种运镜。
5. 同场景相邻镜：色调与光影方向尽量延续；换场才大变色调。

## 示例（合格）
- shot_size: 近景
- angle: 平视略仰
- composition: 男主脸偏右三分，左侧留门框前景
- lighting: 门内暖黄侧光，门外冷白轮廓光
- color_tone: 暖黄主调，冷白辅对比
- motion: 缓慢推镜至下巴以上；男主抬下巴半寸
- transition: soft dissolve
